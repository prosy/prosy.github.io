● RETRIEVAL · DETERMINISTIC RAG

Retrieval I built, not bought.

  • A deterministic, citation-based RAG service over a multimodal corpus — FastAPI / Python, BM25, live on Cloud Run.
  • The LLM interprets the question. It never writes the answer. Ask “how do I change the battery?” and you get Section 6-28, pp. 460–462 — the manual's own words, not a paraphrase of them.
  • Every diagram carries its own page cite. Pages are decomposed into atomic answer-units so a machine can comprehend them.

● REUSABLE PRIMITIVES

I built the primitives, not just the agent.

  • Approve / undo — a one-use cryptographic grant, bounded transaction gates, and human-in-the-loop confirmation. An agent cannot complete a purchase unverified.
  • Context continuity — an owner-scoped user domain compiled just-in-time into context, so the agent picks up where the traveler left off.
  • Explanations — section-level evidence and a confidence band on every answer.
  • Each one is reusable on any surface where an agent acts for someone.
● EVALUATION · FRONTIER MODELS

I ran the test that killed my own thesis.

  • Drove Gemini and Perplexity through 30 questions across two owner's manuals, twice each — 119 answers scored, every citation logged.
  • My going-in thesis was that the citation slot was empty. The data refuted it. Gemini cites nothing by design; Perplexity cites ~98%, and those slots are already owned.
  • So I changed the plan, not the data. Measure first, then build.
